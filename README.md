Tnis is a fully dockerised version for modul 7, cohort 2026 of Data Engineering zoomcamp
https://github.com/DataTalksClub/data-engineering-zoomcamp/tree/main/07-streaming/workshop
1. build docker using now cache option.
<sudo docker compose build --no-cache>
2. run containers
<sudo docker compose up -d>
In this version you will run code using jupyter container instead of local machine, to access it use command 
<sudo docker compose logs jupyter>
Beside that there is no difference in running all commands in the workshop
